# Comparador de Hipotecas

Herramienta web para comparar hasta 8 ofertas de hipoteca fija lado a lado, incluyendo simulación de amortización anticipada.

## Funcionalidades

- Comparación de hasta 8 hipotecas simultáneamente
- Soporte para tipo inicial diferente (ej. tipo reducido los primeros años)
- Cálculo de TAE real incluyendo vinculaciones y comisiones
- Simulación de hasta 3 escenarios de amortización anticipada (reducir plazo o cuota)
- Tabla de amortización detallada (mensual y anual)

## Requisitos

- Node.js 18+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Build de producción

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite
- Tailwind CSS
